# 🎉 Party Game Bot

Telegram-бот для разгара вечеринки: задания, дуэли, голосования, оценки, ачивки и номинации!

## Что умеет

- Регистрация участников с именем и фото
- Админ отмечает именинников (они только голосуют/оценивают)
- **🎲 Новое задание** — случайному участнику приходит угарно-алкогольное задание
- Исполнитель: ✅ Выполнил / ❌ Отказываюсь
- Если выполнил — все голосуют «правда?» → все ставят оценку 1-5 → балл = сумма оценок
- Если отказался → штраф 10-200 ₽ в общак + −3 балла
- **🎰 Рулетка отказа** — 1 раз за вечер можно получить лёгкое задание и отыграть штраф
- **⚔️ Дуэль** — 2 случайных игрока соревнуются в одном задании, все голосуют кто круче (+10 победителю, +2 проигравшему)
- **🎯 Кастомное задание от админа** — админ пишет задание и выбирает кому
- 🎖 **Ачивки** — Первопроходец, Железный, Дуэлянт, Феникс, Актёр, Пятёрочник, Душа партии, Слизень
- **🏁 Финал:** картинка победителя с короной, таблица мест, номинации, золотое задание вечера, общий сбор в общак

## Как запустить

### 1. Создать бота в BotFather
- `/newbot` → выбрать имя и username → получить токен

### 2. Добавить бота в групповой чат и написать `/setgroup`
Бот будет кидать туда объявления.

### 3. Деплой на Railway
- Загрузить репо в GitHub
- Railway: **New Project → Deploy from GitHub Repo → party-game-bot**
- В Variables задать `BOT_TOKEN` (и `ADMIN_ID` если хочешь фиксированного админа, иначе первый написавший `/start` станет админом)

### 4. Локальный запуск
```bash
pip install -r requirements.txt
BOT_TOKEN=твой_токен python3 bot.py
```

## Сценарий игры

1. Все пишут `/start` в ЛС бота → имя → фото
2. Админ в беседе пишет `/setgroup` чтобы бот знал куда кидать анонсы
3. Админ жмёт **⭐ Отметить именинников** → отмечает кого надо
4. Когда все готовы — жмёт **🎲 Новое задание**
5. Случайному не-имениннику прилетает задание в ЛС
6. Исполнитель делает → жмёт «✅ Выполнил»
7. Всем в ЛС: «Правда выполнил?» 👍/👎
8. Админ жмёт **🔒 Закрыть голосование**
9. Если засчитали → всем в ЛС оценки 1-5
10. Админ жмёт **🔒 Закрыть оценки**
11. Баллы начислены, в общий чат идёт результат
12. Цикл повторяется (можно и дуэли, и кастомные)
13. В конце админ жмёт **🏁 Завершить вечеринку** → бот объявляет победителя

## Системы баллов

- Выполненное задание → сумма оценок всех голосующих (1-5 от каждого)
- Отказ → −3 балла + штраф 10-200 ₽ в общак
- Дуэль: победитель +10, проигравший +2, ничья по +5
- Рулетка отказа: +4 балла (возвращение штрафа)

## Файлы

- `bot.py` — основной код
- `tasks.py` — списки заданий (обычные, лёгкие для рулетки, дуэльные)
- `bot_data.json` — локальное хранилище (создаётся автоматически)
- `group_chat.txt` — сохранённый ID групп-чата
